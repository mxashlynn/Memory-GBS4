## Variables

- Game State
    - 0 = Unstarted
    - 1 = In Play
    - 2 = Lost
    - 3 = Won
- Player Match Count
    - 0…10 = How many card pairs found so far.
- Player Flips Made Count
    - 0…MAX_INT = How many flips made so far.
- Player Flips Left Count
    - 3…0 = How many flips remaining till game over.
