# Memory 🐍🎴👭

A monstergirl matching game for Game Boy.

![very gay](https://img.shields.io/badge/very-gay-ac276c?style=plastic)


## Goals 🗺🏯✨

- To produce a very small game fast.
- To learn how tile swapping works in GB Studio 3.x.
